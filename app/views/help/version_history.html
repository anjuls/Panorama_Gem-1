Releasenotes: Panorama

enhancement within current release

Release 2.1.3 (2015-11-04)

  • Allow stacked charts in diagrams
  • Switch value points on/off in diagrams
  • Show 'others'-chart in Top-10 diagrams with cumulated values for charts outside top 10
  • Cache last used DB-connection in server to suppress disconnect after request and reconnect if next request uses same DB

Release 2.1.1 (2015-10-26)

  • rework look and feel
  • Recognize touch events on tablets and phones
  • Show last analyzed date in execution plan lines
  • Grouping active session history by session-type (foregreound/background)
  • Repeat last called menu action without deep-dive in menu-structures by clicking icon right to menu
  • Active Session History: Show tablespace name as object name if no current object exists. Helps to recognize action in UNDO-TS.
  • List captured bind values also for datatype TIMESTAMP
  • Replace WM_CONCAT with LISTAGG for database release 11.2 and above beacause WM_CONCAT is not supported beginning with 12c
  • Replace sys.User$ with DBA_Users because access on sys.User$ is not part of SELECT ANY DICTIONARY beginning with 12c
  • Dragnet investigation for parallel degradation (FORCE SERIAL)

Release 2.0.79 (2015-07-16)

  • Show load of parallel query slaves within coordinator session in Active Session History
  • Distinguish between active session load executed in parallel query slaves or not in Active Session History
  • List global database triggers like LOGON-Trigger at "DBA General" / "Database triggers"
  • Dragnet investigation: Use filter to search selection in list
  • Dragnet investigation: Further selections for performance pitfalls
  • Object description: show size and state of subpartitions
  • Disk stroage summary: Show size and content of Fast recovery Area
  • Filter string works also for username in Active Session History
  • Show retention for AWR data at start page
  • Add DB-Name and AWR retention in DBID-selection at start page if cloned database has different DB-IDs in AWR history
  • Evaluate history for one SQL-ID over all RAC-instances (not only per instance)
  • Show LOB details in object description, include LOB-size in size calculation of object size
  • Describe database objectes as separate menu item
  • SSL-encrypted network traffic between browser and Panorama-instance (https)
  • SQL-details historic: List complete history of recorded bind values

Release 2.0.30 (2015-01-22)

  • Historic timeline resource usage and resource limits
  • Historic timeline for real CPU-usage and DB-time
  • Correct calculation of CPU-Time and DB-Time in listing of active session history

Release 2.0.3 (2014-09-17)

  • Keep DB connection open after request for reuse in next request without reconnect

Release 1.13 (15.01.2014)
